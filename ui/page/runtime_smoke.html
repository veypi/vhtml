Runtime Smoke

这页不是文档，而是运行时冒烟检查页。它只覆盖浏览器里可以直接观察的链路：组件生命周期、slot 投递、缓存页激活。`scoped` 仍然依赖模块响应头或模块入口环境注入，这里只做约束说明，不做伪验证。

scoped 约束

当前静态站点不能完整覆盖模块隔离验证，因为 `env.scoped` 需要模块入口或响应头注入。这里保留的是你这次重构后的预期规则，便于手工对照。

scoped = /xxx/auth
组件 HTML: /xxx/auth/feature/panel.html
$axios baseURL: https://host/xxx/auth
接口请求: /api/profile -> https://host/xxx/auth/api/profile
