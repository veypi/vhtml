介绍

vhtml 是一个轻量级响应式前端框架，采用原生 HTML5 语法，无需复杂的构建工具即可快速开发 Web 应用。

特性

  • 轻量 - 压缩后仅 96KB，无外部依赖
  • 简单 - 基于标准 HTML5，学习成本低
  • 响应式 - 自动数据绑定，状态驱动视图
  • 组件化 - 支持组件封装和复用
  • 路由 - 内置客户端路由系统

浏览器支持

vhtml 支持所有现代浏览器，包括 Chrome、Firefox、Safari、Edge 等。

快速开始

安装

通过 npm 安装：

npm install @veypi/vhtml

或直接通过 CDN 引入：

<script src="https://unpkg.com/@veypi/vhtml/dist/vhtml.min.js"></script>

创建第一个页面

创建 index.html：

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>我的应用</title>
  </head>
  <body>
    <h1>{{ message }}</h1>
    <button @click="updateMessage">点击</button>
  </body>
  <script setup>
    message = 'Hello vhtml!';
    updateMessage = () => {
      message = '你好，世界！';
    };
  </script>
</html>

目录结构

推荐的 vhtml 项目目录结构：

project/
├── ui/
│   ├── root.html          # 应用根页面
│   ├── app.js             # 应用配置
│   ├── routes.js          # 路由配置
│   ├── assets/
│   │   └── global.css     # 全局样式
│   ├── layout/
│   │   └── default.html   # 默认布局
│   ├── page/
│   │   ├── index.html     # 首页
│   │   └── 404.html       # 404页面
│   └── components/        # 组件目录
└── ...

模板语法

文本插值

<p>{{ message }}</p>

属性绑定

<!-- 动态属性 -->
<a :href="url">链接</a>

<!-- 双向绑定 -->
<input v:value="inputValue">

条件渲染

<div v-if="type === 'A'">A</div>
<div v-else-if="type === 'B'">B</div>
<div v-else>其他</div>

列表渲染

<div v-for="(item, index) in list">
  {{ index }}: {{ item.name }}
</div>

响应式数据

声明数据

在 <script setup> 中使用 = 直接赋值声明响应式数据：

<script setup>
  // 基础类型
  count = 0;
  message = 'Hello';

  // 对象和数组
  user = { name: '张三', age: 25 };
  items = ['a', 'b', 'c'];
</script>

访问数据

在模板中直接使用变量名：

<p>{{ message }}</p>
<p>{{ user.name }}</p>

在 script 中访问

<script>
  // 使用 $data 访问和修改
  $data.count = 10;
  $data.message = '新消息';
</script>

事件处理

监听事件

<button @click="handleClick">点击</button>
<input @input="onInput">

事件处理函数

<script setup>
  handleClick = () => {
    console.log('按钮被点击');
  };

  onInput = (event) => {
    console.log('输入值:', event.target.value);
  };
</script>

触发自定义事件

<script>
  // 向父组件触发事件
  $emit('custom_event', data);
</script>

组件

创建组件

在 /ui/components/ 目录下创建 HTML 文件：

<!-- /ui/components/my-button.html -->
<button @click="onClick">{{ label }}</button>
<script setup>
  label = '按钮';
  onClick = () => {
    $emit('click');
  };
</script>

使用组件

路径 /ui/components/my-button.html 对应标签 <components-my-button>：

<components-my-button
  :label="'提交'"
  @click="handleSubmit">
</components-my-button>

路由

配置路由

在 /ui/routes.js 中配置：

const routes = [
  { path: '/', component: '/page/index.html', name: 'home' },
  { path: '/about', component: '/page/about.html', name: 'about' },
  { path: '*', component: '/page/404.html', name: '404' },
];

export default routes;

导航

<!-- 链接导航 -->
<a href="/about">关于</a>

<!-- 编程导航 -->
<script>
  $router.push('/about');
  $router.back();
</script>

API 调用

vhtml 内置 $axios 用于 HTTP 请求：

GET 请求

<script>
  $axios.get('/api/users')
    .then(data => {
      console.log('用户列表:', data);
    })
    .catch(error => {
      $message.error('获取失败');
    });
</script>

POST 请求

<script>
  $axios.post('/api/users', { name: '张三', age: 25 })
    .then(data => {
      $message.success('创建成功');
    });
</script>

插槽

默认插槽

<!-- 组件内 -->
<vslot>
  <p>默认内容</p>
</vslot>

<!-- 使用时 -->
<my-component>
  <div>自定义内容</div>
</my-component>

命名插槽

<!-- 组件内 -->
<vslot name="header">
  <h1>默认标题</h1>
</vslot>

<!-- 使用时 -->
<my-component>
  <div vslot="header">
    <h1>自定义标题</h1>
  </div>
</my-component>
