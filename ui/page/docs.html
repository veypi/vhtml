{{ $t('docs.content.api.title') }}

{{ $t('docs.content.api.desc') }}

运行时变量池固定为 $sys > $data > $ctx > $mod。其中系统能力来自 $sys，模块能力来自 $mod。

{{ $t('docs.content.api.get') }}

<script>
            $mod.$axios.get('/api/users')
            .then(data => {
            console.log('Users:', data);
            })
            .catch(error => {
            $message.error('{{ $t('docs.content.api.error') }}');
            });
            </script>

{{ $t('docs.content.api.post') }}

<script>
            $mod.$axios.post('/api/users', { name: 'User', age: 25 })
            .then(data => {
            $message.success('{{ $t('docs.content.api.success') }}');
            });
            </script>
